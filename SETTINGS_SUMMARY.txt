**Storage Format** (localStorage: 'lottery_settings'):
```json
{
  "effects": {
    "snow": true,
    "fireworks": false,
    "bubbles": false,
    "rainbow": false,
    "stars": false,
    "particles": false

    "id": "video-id|image-id",
    "path": "/static/path/to/file"
  }
}
```

**Key Functions**:
- `InitializeSettings()` - Load on modal open
- `EnableEffect(effect)` - Activate effect
- `DisableEffect(effect)` - Deactivate effect
- `ApplySettings(settings)` - Apply all settings
- `SaveSettings()` - Persist to localStorage
- `ResetSettings()` - Revert to defaults
